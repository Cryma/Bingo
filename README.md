# Just Bingo

## Installation

#### Install dependencies

```sh
composer install
```

```sh
yarn install
```

#### Project setup

```sh
cp .env.example .env
```

```sh
php artisan key:generate
```

```sh
php artisan migrate
```

## Testing

```sh
./vendor/bin/phpunit
```
